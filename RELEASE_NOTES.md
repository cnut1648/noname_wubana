## 五班阿扩展

五班同学主题 DIY 武将扩展（基于无名杀）。共 **51 名武将 / 约 133 个技能**，均为同学梗自制。

### 本次更新（v1.2）
- 新增 **研 / 工 / 游** 三个新势力，共 **9 名武将 / 28 个技能**：
  - **研**：研杜时宇（实验·百川·勃朗）、研徐家澍（小本生意·转运·走私·FBI）、研李博为（苦寒·自闭·向阳）
  - **工**：工大黄（睹物·思人·出差）、工宋轶健（调班·白班·夜班）
  - **游**：游宋轶健（同质·赤史·战术）、游李博为（长夜·火把·会赢的）、游杜时宇（延迟·巧匠·赢你妈）、游徐家澍（创新·黑人拍皮球·篱下）
- 含使命技、协力技、势力技（阳/阴）等新机制。

### 下载
- **Windows**：`noname.Setup.*.exe` — 安装版，已内置全部素材与扩展，安装后即可游玩。
- **安卓**：`noname-wuban-test-*.apk` — **非官方测试包**，需侧载安装；体积较大（约 1.2GB，自带素材可离线玩）。
- `noname.full.zip` / `noname.core.zip` — 完整网页包 / 纯代码包（进阶/开发用途）。

- 扩展**默认已开启**，进入游戏后在选将界面即可看到「五班阿」武将。
